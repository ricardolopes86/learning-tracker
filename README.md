## How to run it

Clone it using 

    $ git clone git@github.com:ricardolopes86/learning-tracker.git

Then just `cd` into the `learning-tracker/server` folder and run the following commands:

    $ npm install
    $ npm install -g nodemon
    $ npm run dev
